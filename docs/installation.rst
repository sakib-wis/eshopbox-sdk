Installation
============

Install via pip:

.. code-block:: bash

   pip install eshopbox-sdk

Or install from source:

.. code-block:: bash

   git clone https://github.com/sakib-wis/eshopbox-sdk.git
   cd eshopbox-sdk
   pip install -e .

Requirements
-----------

- Python 3.8+
- requests
- Sphinx (for building docs)
